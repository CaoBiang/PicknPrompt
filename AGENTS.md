# PicknPrompt
一个完善的 Edge 浏览器 AI 对话插件，解决的核心需求为：用户在网页上看到的重要信息可以框选、添加到暂存区，然后再选择暂存区中一个或多个信息作为上下文进行对话。

# 功能模块

1. 大模型配置模块：配置、存储多个大模型信息，每一个信息都包括 URL、API_KEY、Model。
2. 助手配置模块：配置、存储一多个助手，助手本质上是预设Prompt，作为系统提示词传入对话。
3. 对话管理模块：存储、管理多个对话，支持对话导入与导出，遵循markdown语法与渲染。
4. 大模型通信模块：支持最新的OpenAI格式协议。
5. 暂存区管理模块：用户可以在网页上框选文本、图片添加到暂存区（称之为信息），每条信息都记录了信息来源URL与标题。

# 界面要求

1. 素雅灵动，禁止使用高饱和色、渐变色。

# 代码结构

1. 按照模块分离解耦，方便后期维护。

# 用户故事

## 使用方法1
1. 用户安装插件后，先配置一个或多个大模型配置作为大模型的来源。
2. 再配置一个助手（有默认助手）的上下文。
3. 浏览网页的时候，看到有用的信息和图片，框选、拖拽到右侧的暂存区。
4. 用户点开本插件详细的页面，通过关键词搜索、URL或标题筛选暂存区中的内容并勾选，支持导出。
5. 选择一条或多条信息，选择合适的助手进行对话。

## 使用方法2
1. 用户安装插件后，先配置一个或多个大模型配置作为大模型的来源。
2. 再配置一个助手（有默认助手）的上下文，并设置为“快捷助手”。
3. 浏览网页的时候，看到有用的信息和图片，框选，弹出快捷助手列表，点击助手名称直接开始对话。
